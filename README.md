## üîç Description

**42 Piscine Python** is an intensive Python bootcamp at 42 School, structured into 10 progressive modules. Each module covers a fundamental aspect of Python programming, from basic variable types all the way to functional programming and design patterns.

The piscine covers:
- üå± **Modules 01‚Äì02**: Foundations (variables, types, error handling)
- üìä **Modules 03‚Äì04**: Data manipulation (structures, files, serialization)
- üèóÔ∏è **Modules 05‚Äì07**: Object-Oriented Programming (polymorphism, ABC, patterns)

## üíª Usage

Each module's exercises can be run independently:

